  - title: "今日のゴール"
    body: |
      ・MV全体の色調を統一できる
      ・グレイン（ノイズ）で素材を馴染ませられる
      ・最終コンポを正しく組める
      ・YouTube向けの書き出し設定ができる

      → MVとして公開できる“完成形”を作る

  - title: "仕上げ工程の全体像"
    body: |
      最終仕上げ＝“世界観の統一”
      工程：
        1. 色調統一（Color Grading）
        2. ノイズ追加（Grain）
        3. 最終コンポジット
        4. 書き出し設定
        5. サムネイル制作
      → この工程で作品の“完成度”が決まる

  - title: "色調統一（Color Grading）"
    body: |
      色調統一＝“全カットを同じ世界観に揃える”
      ・LUT（Look-Up Table）を適用
      ・カラーバランスで微調整
      ・影／中間／ハイライトを整える
      ・キャラと背景の色温度を合わせる
      → 色調が揃うと“プロの映像”に見える

  - title: “色調統一の実践ポイント”
    body: |
      ・キャラの肌色が破綻していないか
      ・背景の色が強すぎて主役を奪っていないか
      ・サビだけ色を少し明るくするのも有効
      ・全体のコントラストを統一する
      → “世界観の一貫性”が最重要

  - title: “LUTの使い方（実践手順）”
    body: |
      LUT＝色調プリセット。1クリックで映像の雰囲気が変わる
      適用手順：
        1. 最終コンポの最上部に「調整レイヤー」を追加
        2. エフェクト →「LUT適用（Apply Color LUT）」を追加
        3. フリーLUT（例：Ground Control / Koji）を読み込む
        4. 不透明度を30〜60%に下げて”かけすぎ”を防ぐ
      ポイント：
        ・LUTは素材に直接かけない（調整レイヤー経由が鉄則）
        ・LUTなしでもOK → 次スライドの手動調整で代替できる

  - title: “LUTなしでの手動カラーグレーディング”
    body: |
      LUTが合わない場合は「Curves（カーブ）」で調整する：
        ・RGB全体：コントラストをS字カーブで強める
        ・R（赤）：シャドウにわずかに足す → 温かみが出る
        ・B（青）：ハイライトを少し引く → 映画調になる
      Levels（レベル）でも代用可：
        ・出力の最大値を235に下げると柔らかい印象になる
      ガンマシフト対策：
        ・「カラープロファイルをリニアに変更」をOFFにする
        ・書き出し後にQuickTimeでプレビューして確認
      → Curvesだけで”雰囲気のある映像”は十分作れる

  - title: "グレイン（ノイズ）の役割"
    body: |
      グレイン＝“素材を馴染ませる接着剤”
      ・CG特有の“綺麗すぎる質感”を消す
      ・キャラ・背景・エフェクトを統一
      ・0.3〜1.0%の弱いノイズが最適
      注意：
        ・過剰にすると“汚れ”になる
      → グレインは“世界観の統一感”を作る

  - title: "最終コンポ構造"
    body: |
      最終コンポは“レイヤーの役割”を明確にする
      推奨構造：
        ・BG（背景）
        ・MG（中景・パララックス）
        ・FG（前景・装飾）
        ・キャラクター
        ・歌詞（テキスト）
        ・撮影効果（グロー・色収差・シェイク）
        ・カラーグレーディング
        ・グレイン
      → 順番が崩れると画面が破綻する

  - title: "最終チェックポイント"
    body: |
      書き出し前に必ず確認：
        ・音ズレがないか
        ・歌詞の誤字・タイミング
        ・カメラの揺れが強すぎないか
        ・色が破綻していないか
        ・画面の明るさが適切か
        ・パララックスが破綻していないか
      → “違和感ゼロ”が完成の基準

  - title: "書き出し設定（YouTube最適化）"
    body: |
      推奨設定（H.264）：
        ・Format：H.264
        ・Preset：YouTube 1080p
        ・Bitrate：20〜30 Mbps
        ・Frame Rate：29.97 fps
        ・Render at Maximum Depth：ON
        ・Use Maximum Render Quality：ON
      → YouTubeは“高ビットレートほど綺麗に残る”

  - title: "書き出し時の注意点"
    body: |
      ・黒つぶれ／白飛びがないか
      ・音量が大きすぎないか（-14〜-12 LUFS）
      ・色がYouTubeで変わらないか（ガンマシフト対策）
      ・冒頭1秒は黒にしない（クリック率低下）
      → 公開後の見え方を想定する

  - title: "サムネイル制作（MVの顔）"
    body: |
      サムネイルは“作品の入口”
      ・GPTで生成したキャラを使用
      ・明るい色・大きい文字
      ・3〜5語の短いタイトル
      ・目線がカメラを見る構図が強い
      ・AEで文字入れして書き出し
      → サムネイルの出来で再生数が変わる
